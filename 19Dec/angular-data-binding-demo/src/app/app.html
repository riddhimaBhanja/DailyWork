Data Binding Types Summary

  • Interpolation: {{ '{{' }} value {{ '}}' }} - One-way from component to view
  • Property Binding: [property]="value" - One-way from component to view
  • Event Binding: (event)="handler()" - One-way from view to component
  • Two-Way Binding: [(ngModel)]="value" - Two-way between component and view
  • Class Binding: [class.className]="condition" - Conditional CSS classes
  • Style Binding: [style.property]="value" - Dynamic inline styles
  • Attribute Binding: [attr.attribute]="value" - For HTML attributes
